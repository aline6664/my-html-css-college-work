Lista de Definição

HTML
- Bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.
CSS
- Linguagem baseada em regras, especificando grupos de estilo que devem ser aplicados para elementos particulares ou grupos de elementos na sua página web.
PHP
- Linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.
PYTHON
Linguagem de programação amplamente usada em aplicações da Web, desenvolvimento de software, ciência de dados e machine learning (ML).
